# Security Policy

## Supported Versions

This project hasn't published a NuGet release yet (see the README) — until then, only the latest commit on `main` is supported.

## Reporting a Vulnerability

Please report security issues privately rather than opening a public issue: use [GitHub's private vulnerability reporting](https://github.com/JasonGoble/FiveTalents.Calendar/security/advisories/new) for this repository, or email 20996977+JasonGoble@users.noreply.github.com.

Include what you've found, how to reproduce it, and its potential impact. This is a small project maintained outside of working hours, so response times may vary, but reports will be acknowledged and addressed as soon as reasonably possible.
